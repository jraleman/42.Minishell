# 42 Minishell

### Allowed Functions

- malloc, free
- access
- open, close, read, write
- opendir, readdir, closedir 
- getcwd, chdir
- stat, lstat, fstat
-  fork, execve
- wait, waitpid, wait3, wait4
- signal, kill
- exit
